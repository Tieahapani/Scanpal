# ScanPal

AI-powered receipt organizer with automatic vendor detection and expense categorization. Built using **Flutter**, **Azure OCR**, and **Gemini AI**. Features intelligent receipt scanning, automatic compartment organization, expense analytics, and local storage.

## Problem Statement

Managing physical receipts presents several challenges for individuals tracking personal expenses. Receipts accumulate and become disorganized, locating specific receipts requires significant time, manual expense categorization is labor-intensive, and deriving spending insights requires additional spreadsheet work. **ScanPal** addresses these challenges through automated receipt processing and intelligent organization.

## Solution Overview

ScanPal automates receipt management from capture to analysis. The application scans receipts, extracts relevant information, identifies vendors, categorizes expenses, and organizes receipts into vendor-specific compartments without manual intervention.

## Features

### Automated Receipt Processing
- Captures receipt images through mobile camera
- Extracts text data using **Azure OCR** technology
- Identifies vendor names using **Gemini AI**
- Categorizes expenses automatically (**Food**, **Shopping**, **Travel**, **Gas**, **Entertainment**)

### Intelligent Organization System
- Creates vendor-specific compartments on first scan
- Routes subsequent receipts to existing compartments automatically
- Eliminates manual sorting and folder management

### Expense Analytics
- **Monthly** spending summaries with category breakdowns
- **Weekly** spending trend analysis
- **Annual** spending pattern visualization
- Vendor spending distribution
- Category-wise expense tracking

### Local Data Storage
- Receipt data stored locally using **Isar database**
- Complete offline functionality after initial setup
- Privacy-focused design with on-device storage
